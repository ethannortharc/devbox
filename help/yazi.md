# yazi — Terminal File Manager

## Navigation
  h j k l    Navigate (parent/down/up/enter)
  g g        Go to top       G  Go to bottom
  H L        Back/forward in history
  ~          Go to home      /  Search

## File Operations
  o          Open file
  Enter      Open / enter directory
  y          Yank (copy)     p  Paste
  d          Trash           D  Permanently delete
  r          Rename          a  Create file
  Space      Select          V  Visual select

## View
  .          Toggle hidden files
  s          Sort menu
  1 2 3      View modes
  Tab        Switch pane
  z          zoxide jump (fuzzy cd)

## Quick Actions
  q          Quit
  :          Command mode
  Ctrl+s     Shell command in current dir
